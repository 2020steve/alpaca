Welcome to Alpaca Project!

Introduction

Alpaca is a browser-side forms and templating engine which gives your web site the ability to render elegant content entry and presentation forms.

Designed for rendering forms based on JSON data or JSON schema, a universal lightweight data-interchange format, Alpaca can be easily embedded into your web page or web application no matter what platform or device you are using.

Alpaca also provides optional components for storing, retrieving, versioning and auditing JSON data and schema to or from Gitana Cloud Repository.
