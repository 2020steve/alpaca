Alpaca via AMD

Alpaca also comes packaged as an AMD (Asynchronous Module Definition) file. If you're building large web application projects, we strongly suggest considering AMD as it helps to make dependency management much easier.

Quick Start

An empty Alpaca project that you can use as a template for building your own forms. All of the dependencies are loaded via require() and define() methods. Adjust the source code to fit your own schema, options and layout needs.

Registration

A sample registration form that uses a custom layout and custom validation messages. The results of the form are posted to a server-side PHP file for capture into a database. The storage into the DB is left as an exercise for the end user.
