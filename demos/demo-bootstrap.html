Alpaca with Twitter Bootstrap

Alpaca natively renders into Twitter Bootstrap. Bootstrap is a UI framework that provides responsive design for your web sites. It also provides convenient formatting for a number of HTML elements.

These projects include the alpaca-bootstrap.css file to provide compatibility with Twitter Bootstrap.

Quick Start

An empty Alpaca project that you can use as a template for building your own forms. Adjust the source code to fit your own schema, options and layout needs.

Registration

A sample registration form that uses a custom layout and custom validation messages. The results of the form are posted to a server-side PHP file for capture into a database. The storage into the DB is left as an exercise for the end user.
