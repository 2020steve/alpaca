Alpaca with jQuery UI

Alpaca supports jQuery UI along with jQuery UI themes. JQuery UI is an excellent library that rides right on top of jQuery to provide beautiful user interfaces. Alpaca renders nicely into jQuery UI.

These projects include the alpaca-jqueryui.css file to provide compatibility with jQuery UI.

Quick Start

An empty Alpaca project that you can use as a template for building your own forms ready-to-run in the browser. All of the Alpaca dependencies are loaded using SCRIPT tags. Adjust the source code to fit your own schema, options and layout needs.

Quick Start (List View)

This is a version of the Quick Start project that uses a list view for rendering. The view is set to VIEW_JQUERYUI_CREATE_LIST which is a jQuery UI version of the list view in create mode. As a result, labels are rendered side-by-side with value fields.

Registration

A sample registration form that uses a custom layout and custom validation messages. The results of the form are posted to a server-side PHP file for capture into a database. The storage into the DB is left as an exercise for the end user.
