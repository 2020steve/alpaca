Alpaca Demos

This page provides a listing of pre-built demos for Alpaca.

  • Quick Start (Browser)
    An empty Alpaca project that you can use as a template for building your own forms ready-to-run in the browser. All of the Alpaca dependencies are loaded using SCRIPT tags. Adjust the source code to fit your own schema, options and layout needs.
  • Registration (Browser)
    A sample registration form that uses a custom layout and custom validation messages. The results of the form are posted to a server-side PHP file for capture into a database. The storage into the DB is left as an exercise for the end user.
  • Registration with Bootstrap (Browser)
    The same sample registration form using Twitter Bootstrap for its user interface. Twitter Bootstrap is a really nice and lightweight responsive UI framework. Alpaca renders Twitter Bootstrap friendly forms for your end users.

Getting Started

These demos ship with the Alpaca source code. Each demo is standalone and utilizes JS files that are stored locally. To update each demo's dependencies, simply run ant.
