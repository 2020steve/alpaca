What is Alpaca?

Alpaca is a plugin for jQuery, jQuery UI and jQuery Mobile that makes it easy to render beautiful HTML5 Web Forms for web and mobile applications.

Alpaca is built around JSON-Schema for expression of the form object model, types, properties, validation logic and constraints. It provides a comprehensive and extensible control library with support for wizards, layout, I18N and pluggable templating engines (like EJS and jQuery Templating).

Alpaca is an open-source, community-led project licensed under Apache 2.0.

How do I use Alpaca?

Copy the following HTML into your web page's <head/> block. 

<script type="text/javascript" src="http://code.jquery.com/jquery-latest.min.js"></script>
<script type="text/javascript" src="http://www.alpacajs.org/js/alpaca-latest.min.js"></script>
<link type="text/css" href="http://www.alpacajs.org/css/alpaca-latest.min.css" rel="stylesheet"/>

Add a <div> tag to your HTML page and call $.alpaca() with your form data. 

<div id="form1"></div>

<script type="text/javascript">
$(function() {
  $("#form1").alpaca({
    "schema": {
      "title":"User Feedback",
      "description":"What do you think about Alpaca?",
      "type":"object",
      "properties": {
        "name": {
          "type":"string",
          "title":"Name"
        },
        "ranking": {
          "type":"string",
          "title":"Ranking",
          "enum":['excellent','not too shabby','alpaca built my hotrod']
        }
      }
    }
  });
});
</script>

Enjoy your newly rendered Alpaca form.

That's all there is to it.

Alpaca lets you provide data, schema and layout information using simple JSON.
